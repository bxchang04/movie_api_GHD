Documentation

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of ALL movies to the user /movies GET None A JSON object holding data about all the movies
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing a title, description, genre property, director property and image. Example:

{ "Genre": { "Name": "Thriller", "Description": "Thriller film, also known as suspense film or suspense thriller, is a broad film genre that involvesexcitement and suspense in the audience." }, "Director": { "Name": "Jonathan Demme", "Bio": "Robert Jonathan Demme was an American director, producer, and screenwriter.", "Birth": "1944", "Death": "2017" }, "Actors": [], "_id": "5d9bab4307ab30237c44c353", "Title": "Silence of the Lambs", "Description": "A young FBI cadet must receive the help of an incarcerated and manipulative cannibal killer to help catch another serial killer.", "ImagePath": "silenceofthelambs.png", "Featured": true }

Get data about a movie genre /movies/genres/[Name] GET None A JSON object holding data about a movie genre. Example:

{ "Name": "Action", "Description": "Action film is a film genre in which the protagonist or protagonists are thrust into a series of events that typically include violence, extended fighting, physical feats, and frantic chases." }

Get data about a director /movies/directors/[Name] GET None A JSON object holding data about a director. Example:

{ "Name": "Jon Favreau", "Bio": "Jonathan Favreau is an American actor, director, producer, screenwriter, and chef.", "Birth": "1966" }

Add new user accounts /users POST A JSON object holding data about the user to add, structured like:

{ "Username" : "helloworld123456", "Password" : "password123456", "Email" : "example123456@gmail.com", "Birthday" : "1970-03-21" }

A JSON object holding data about the user that was added, including an ID:

{ "FavoriteMovies": [], "_id": "5d9d82069bc166069f68524d", "Username": "helloworld123456", "Password": "password123456", "Email": "example123456@gmail.com", "Birthday": "1970-03-21T00:00:00.000Z", "__v": 0 }

Update a user's information /users/[Username] PUT A JSON object holding data about the user to add, structured like:

{ "Username" : "Newusername1231561", "Password" : "Newpassword3486198", "Email" : "Newemail16513651@gmail.com", "Birthday" : "1980-10-15" }

A text message indicating the user's information was updated
Add movies to user's list of favorites /users/[Username]/Movies/[MovieID] POST None A message indicating the movie was added to the user's list of favorites
Remove movies from user's list of favorites /users/[Username]/Movies/[MovieID] DELETE None A message indicating the movie was removed from the user's list of favorites
Remove user accounts /users/[Username] DELETE None A message indicating the user's account was deleted
